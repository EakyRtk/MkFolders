# MkFolders
<img src="addons/MkFolders/icon.svg" alt="MkFolders Icon" width="64" height="64">
**MkFolder** is a tool to create multiple folders at once. Save and load folder names template to quickly create your working space.

## Features
- Open MkFolder with a shortcut, **CTRL+SHIFT+M** or in the **Tools** section in the editor
- Select multiple folders (player, enemy, etc.) and create your folders for each (textures, scripts, sound, etc.)
- Save or load your folder structure templates (.txt) to quickly create your working environment
- Nested folder creation (assets/player/sounds etc.)
- Undo/Redo Support 

## Installation

1. Clone or download the repository into `res://addons/MkFolders/`
2. Enable the plugin in **Project > Project Settings > Plugins**

## Usage

1. Select the folder(s) you want to work on, if none selected it works on the root "res://" directory.
2. Open the plugin dialog with the **shortcut (CTRL + SHIFT + M)** or from the **Project > Tools**
3. Enter folder names separated by lines
4. Optionally select a template or save the current input as a template
5. Press "MAKE THEM!" or use **Ctrl+Enter** to create the folders
